Les navigateurs internet peuvent bloquer les sites web pour les empêcher d'émettre des notifications sonores. Il s'agit d'une fonctionnalité de sécurité visant à empêcher les bruits indésirables.

Lorsque les notifications sonores sont bloquées par votre navigateur, OrderLine affiche une popup rouge. Vous pouvez cliquer sur cette popup pour activer les notifications sonores pour la session en cours. Ce comportement est documenté dans le Guide de dépannage, section [Popup de notification sonore désactivée](/apps/orderline/depannage/#popup-de-notification-sonore-d-sactiv-e).

Pour éviter de le faire à chaque fois que vous lancez OrderLine, vous pouvez configurer votre navigateur pour maintenir les notifications sonores activées en permanence pour OrderLine.

## Navigateurs de bureau

Si vous utilisez OrderLine sur un appareil de bureau, suivez les étapes correspondantes à votre navigateur.

**Chrome :**

1. Ouvrez Chrome et naviguez vers OrderLine.
2. Cliquez sur l'icône de verrouillage située à gauche de l'URL.
3. Dans le menu déroulant, sélectionnez **Paramètres du site**.
4. Trouvez le paramètre de **Son** et changez-le en **Autorisé**.

**Firefox :**

1. Ouvrez Firefox et naviguez vers OrderLine.
2. Cliquez sur l'icône semblable à un engrenage juste à gauche de l'URL dans la barre d'adresse.
3. Dans le panneau qui apparaît, recherchez **Lecture automatique** et changez-le en **Autoriser l'audio et la vidéo**.

**Safari :**

1. Ouvrez Safari et naviguez vers OrderLine.
2. Ouvrez le menu **Préférences** et rendez-vous dans l'onglet **Sites**.
3. Sélectionnez **Lecture automatique** dans la barre latérale gauche.
4. Trouvez OrderLine dans la liste et changez la configuration pour **Autoriser toute Lecture Automatique**.

## Navigateurs mobiles

Suivez ces étapes si vous utilisez Chrome sur un appareil mobile.

**Chrome :**

1. Ouvrir OrderLine dans Chrome.
2. Cliquez sur l'icône de verrouillage située à gauche de l'URL.
3. Dans le menu déroulant, sélectionnez **Paramètres du site**.
4. Dans la prochaine écran, vous trouverez une option relative au **Son**. Cliquez dessus pour la changer en **Autoriser**.
